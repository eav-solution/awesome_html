⚠️ Camera đang khởi động

Để camera hoạt động, bạn PHẢI chạy file qua local server:

  1. Mở Terminal tại thư mục chứa file HTML
  2. Chạy: python3 -m http.server 8000
  3. Truy cập: http://localhost:8000
  4. Cho phép truy cập camera khi trình duyệt hỏi

Trong khi chờ, dùng phím 1, 2, 3, 4 để test hiệu ứng hạt.
